Assignment 1 - Javascript and the DOM


Create a web page that uses Javascript to interact with the DOM (document object model) in the following ways:


  1. 1. Manipulate at least one CSS property through the DOM.
  2. 2. Manipulate at least one other DOM property that is not a CSS property.
  3. 3. Respond to at least two different types of events using event listeners.
