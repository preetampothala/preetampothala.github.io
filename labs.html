Assignment 1 - Javascript and the DOM


Create a web page that uses Javascript to interact with the DOM (document object model) in the following ways:


  1. 1. Manipulate at least one CSS property through the DOM.
  2. 2. Manipulate at least one other DOM property that is not a CSS property.
  3. 3. Respond to at least two different types of events using event listeners.

Assignment 2 - HTML Forms


Create a web page with a form that does the following:


  1. 1. Use at least one <button>
  2. 2. Use at least two <input /> tags with different types
  3. 3. In addition, use at least two of the following other form controls:
    • a. Text box
    • b. Text area
    • c. Check boxes
    • d. Radio buttons
    • e. List box
  5. 4. Every form element must be within a <form> parent tag. There must be at least one <form> tag in the page.
  6. 5. The button should use the input from the other elements to manipulate the page.
